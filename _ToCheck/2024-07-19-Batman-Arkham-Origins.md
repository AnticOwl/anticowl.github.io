## Optional
* ReShade 

## Installation 
Unzip, copy and paste the `dinput8.dll` file into the game folder direcotry `.\Batman Arkham Origins\SinglePlayer\Binaries\Win32`.
The installation directory will depend of the store version (Steam, GOG or Epic).
